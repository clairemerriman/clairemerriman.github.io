Definition 1. Number Systems.

We will use the following number systems and abbreviations:
  • The integers, written \(\Z\text{,}\) is the set \(\{\dots,-3,-2,-1,0,1,2,3,\dots\}\text{.}\)
  • The positive integers are \(\{1,2,3,\dots\}\text{,}\) while the nonnegative integers are \(\{0,1,2,3,\dots\}\text{.}\)
  • The natural numbers, written \(\N\text{.}\) Most elementary number theory texts either define \(\N\) to be the positive integers or avoid using \(\N\text{.}\) Some mathematicians include \(0\) in \(\N\text{,}\) especially logicians.
  • The real numbers, written \(\R\text{.}\)
  • The integers modulo \(n\text{,}\) written \(\Z_n\text{.}\) We will define this set in Strayer Chapter 2, although Strayer does not use this notation.
We will also use the following notation:
  • The symbol \(\in\) means ``element of" or ``in." For example, \(x\in\Z\) means ``\(x\) is an element of the integers" or ``\(x\) in the integers."
