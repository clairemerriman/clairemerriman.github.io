Definition 140.

Let \(p\) be an odd prime number and let \(a\in\Z\) with \(p\nmid a\text{.}\) The Legendre symbol, denoted \(\legendre{a}{p}\text{,}\) is
\begin{equation*} \legendre{a}{p}= \begin{cases} 1, \amp \text{ if $a$ is a quadratic residue modulo $p$} \\ -1, \amp \text{ if $a$ is a quadratic nonresidue modulo $p$} \end{cases} \end{equation*}
