Theorem 173.

The Diophantine equation \(x^n+y^n=z^n\) has a solution no solutions for \(n\geq 3\) if and only if there are no solutions for \(n\) an odd prime or \(n=4\text{.}\)

Proof.

Let \(n\in\mathbb{Z}\) and \(n\geq 3\text{.}\) Let \(n=ab\) where \(a,b\in\mathbb{Z}\) and \(b\) is either an odd prime or \(4\text{.}\) If \(x,y,z\) is a solution to \(x^n+y^n=z^n\text{,}\) then \(x^a,y^a,z^a\) is a solution to \(x^b+y^b=z^b\text{.}\) By contraposition, if \(x^b+y^b=z^b\) has no solutions, then \(x^n+y^n=z^n\) has no solutions.
