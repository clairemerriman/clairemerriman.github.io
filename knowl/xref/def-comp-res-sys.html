Definition 57. complete residue system.

Let \(a,m\in\Z\) with \(m\gt 0\text{.}\) We call the set of all \(b\in\Z\) such that \(a\equiv b \pmod{m}\) the equivalence class of \(a\text{.}\) A set of integers such that every integer is congruent modulo \(m\) is called a complete residue system modulo \(m\text{.}\)
