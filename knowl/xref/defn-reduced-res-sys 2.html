Definition 75. Reduced residue system modulo \(m\).

Let \(m\) be a positive integer. We say that \(\{r_1,r_2,\dots,r_k\}\) is a reduced residue system modulo \(m\) if
  • \((r_i,m)=1\) for all \(i=1,2,\dots,k,\)
  • \(r_i\not \equiv r_j \pmod {m}\) when \(i\neq j,\)
  • for all \(a\in\Z\) with \((a,m)=1,\) \(a\equiv r_1\pmod{p}\) for some \(i=1,2,\dots,k.\)
