    Subsection 8.3.2 Primitive roots and comparing Strayer to the reading

    Definition 123. primitive root.

    Let \(r,m\in\Z\) with \(m\gt 0\) and \((r,m)=1.\) Then \(r\) is said to be a primitive root modulo \(m\) if \(\ord_m r=\phi(r).\)
    We saw in the reading that primitive roots always exist modulo a prime. What about composites?

    Example 124.

    • Since \(\phi(4)=2,\) and \(\ord_4 3=2,\) \(3\) is a primite root modulo \(4.\) The powers \(\{3^1, 3^2\}\) are a reduced residue system modulo \(4.\)
    • Since \(\phi(6)=\phi(3)\phi(2)=2\) and \(\ord_6 5=2,\) \(5\) is a primitive root modulo \(6.\) The powers \(\{5^1, 5^2\}\) are a reduced residue system modulo \(6.\)
    • There are no primitive roots modulo \(8.\) By Theorem 3.3, \(\phi(8)=4.\) Since every odd number squares to \(1\) modulo \(8,\) \(\ord_8 1=1\) and \(\ord_8 3=\ord_8 5=\ord_8 7=2.\)
    • Since \(\phi(9)=3^1(3-1)=6\) by Theorem 3.3, we check:
      \begin{align*} \amp 2^1=2, \amp 2^2=4, \amp\amp 2^3=8\\ \amp 2^4 \equiv7\pmod{9}, \amp 2^5\equiv 5\pmod{9}, \amp\amp 2^6 \equiv 1\pmod{9} \end{align*}
      So \(2\) is a primite root modulo \(9,\) but are there more?
      \begin{align*} 4^1\amp =4, \amp 4^2\amp =2^4\equiv 7\pmod{9}, \amp 4^3\amp =2^6\equiv1\pmod{9} \end{align*}
      We can also use exponent rules and Proposition 5.2 to simplify some calculuations. For example, \(5\equiv 2^5\pmod{9},\) so \(5^i\equiv 2^{5i}\equiv 2^j\pmod{9}\) if and only if \(5i\equiv j\pmod{6}.\)
      \begin{align*} 5^1\amp \equiv 5\pmod{9}, \amp 5^2\amp \equiv 2^{10}\equiv 2^4\equiv 7\pmod{9}, \amp 5^3\amp \equiv 2^{15}\equiv 2^3\equiv 8\pmod{9},\\ 5^4\amp \equiv 2^{20}\equiv 2^2\equiv 4\pmod{9}, \amp 5^5\amp \equiv 2^{25}\equiv 2^1\equiv 2\pmod{9}, \amp 5^6\amp \equiv 1\pmod{9}, \end{align*}
      \begin{align*} 7^1\amp \equiv (-2)\equiv 7\pmod{9}, \amp 7^2\amp \equiv (-2)^2\equiv 4\pmod{9}, \amp 7^3\amp \equiv (-2)^{3}\equiv -8 \equiv 1\pmod{9} \end{align*}
      \begin{align*} \ord_9(1)\amp =1\\ \ord_9(2)\amp =\ord_9(5)=6\\ \ord_9(4)\amp =\ord_9(7)=3\\ \ord_9 (8)\amp =2 \end{align*}
      Thus, \(2\) and \(5\) are primitive roots modulo \(9.\)
    This gives evidence for the fact the if a primitive root exists modulo \(m,\) there are \(\phi(\phi(m))\) primitive roots modulo \(m,\) which we know is true modulo primes.

    Proposition 125. Proposition 5.3.

    Let \(r\) be a primitive root modulo \(m.\) Then
    \begin{equation*} \{r,r^2,\dots,r^{\phi(m)}\} \end{equation*}
    is a set of reduced residues modulo \(m.\)
    This is the general version of Proposition 10.3.2, using exponents \(1,2,\dots,\phi(m)\) instead of \(0,1,\dots,\phi(m)-1\text{.}\) Since Strayer’s statement of Proposition 5.2 is already stated and proved for composites, and both lists have the same number of elements, the only changes to the proof is replacing \(p-1\) with \(\phi(m).\) Note \(a^0\equiv a^{\phi(m)}\equiv 1\pmod{m}\) when \((a,m)=1.\)

    Proposition 126. Proposition 5.4.

    Let \(a,m\in\Z\) with \(m\gt 0\) and \((a,m)=1.\) If \(i\) is a positive integer, then
    \begin{equation*} \ord_m(a^i)=\frac{\ord_m a}{\gcd(\ord_m a,i)}. \end{equation*}

    In Class Activity 15.

    Use only the results through Proposition 5.3 or Lemma 10.3.5 to prove the primitive root version:
    Proposition 127.
    Let \(r,m\in\Z\) with \(m\gt 0\) and \(r\) a primitive root modulo \(m.\) If \(i\) is a positive integer, then
    \begin{equation*} \ord_m(r^i)=\frac{\phi(m)}{\gcd(\phi(m),i)}. \end{equation*}
    Solution.
    Let \(i,r,m\in\Z\) with \(i,m\gt 0\) and \(r\) a primitive root modulo \(m.\) Then \(\ord_m r=\phi(m)\) by definition. Let \(d=(\phi(m),i).\) Then there exists positive integers \(j,k\) such that \(\phi(m)=dj,i=dk\) and \((j,k)=1\) by Proposition 1.10. Then using the proceding equations and exponent rules, we find
    \begin{equation*} (a^i)^j=(a^{dk})^{\phi(m)/d}=(a^{\phi(m)})^k\equiv 1\pmod{m} \end{equation*}
    since \(a^{\phi(m)}\equiv 1\pmod{p}\) by definition. Proposition 5.1 says that \(\ord_p(a^i)\mid j.\)
    Since \(a^{i\ord_p(a^i)}\equiv(a^i)^{\ord_p(a^i)}\equiv 1\pmod{p}\) by definition of order, Proposition 5.1 says that \(\ord_p a\mid i \ord_p(a^i).\) Since \(\ord_p a=\phi(m)=dj\) and \(i=dk,\) we have \(dj\mid dk \ord_p(a^i)\) which simplifies to \(j\mid k \ord_p(a^i)\text{.}\) Since \((j,k)=1,\) we can conclude \(j\mid \ord_p(a^i)\text{.}\)
    Since \(\ord_p(a^i)\mid j, j\mid \ord_p(a^i)\) and both values are positive, we can conclude that \(\ord_p(a^i)=j.\) Finally, we have
    \begin{equation*} \ord_p(a^i)=j=\frac{\phi(m)}{d}=\frac{\phi(m)}{(\phi(m),i)}. \end{equation*}
